¿Que es el código Morse?

El código Morse es un sistema de codificación de caracteres que permite que sus usuarios puedan enviar mensajes utilizando pulsaciones eléctricas cortas o largas, o dicho de otro modo, en forma de puntos y rayas. El código Morse es un método utilizado en telecomunicaciones para codificar caracteres de texto como secuencias estandarizadas de dos duraciones de señal diferentes, llamadas puntos y rayas.

¿Quien invento el código Morse?

Samuel F. B. Morse y su asistente, Alfred Vail, son conocidos como los inventores del código Morse. Este sistema se desarrolló alrededor de 1830 y fue evolucionando en la década siguiente.

¿Cuándo se patentó el código Morse?

Samuel Morse recibió la patente estadounidense, US1647A, para señales telegráficas de puntos y rayas el 20 de junio de 1840. Por otro lado, algunas fuentes afirman que Samuel Morse recibió una patente emitida por el sultán otomano Abdulmejid I para el código Morse. Sin embargo, según las memorias de Cyrus Hamlin y el obituario de The New York Times publicado el 3 de abril de 1872, Samuel Morse no recibió una patente, sino una orden del Imperio Otomano, la Orden de la Gloria.
